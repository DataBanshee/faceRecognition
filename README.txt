A face Recognition Machine Learning Model using different learning methods (Kmeans, Knn) with Dimension Reduction using PCA.

Please provide your own set of training data in a folder called  "faceDatabase" (the images should have a .jpg extension however
you can change these settings by modifying line 3 of faceTracking.m file.

From:
ImageData = ReadImgs('faceDatabase','*.jpg');

To:
ImageData = ReadImgs('your_data_folder','*.image_extension');

Have fun!
